# 🛡️ CyberShield — Multilingual Bullying Detection with Explainable AI

CyberShield is an AI-powered system that detects bullying content across multiple languages and provides explainable insights using model interpretation techniques.

---

## 🚀 Features

* 🌐 Multilingual support (English, Hindi, Telugu, Tamil, Malyalam, Kannada)
* 🤖 Transformer-based model (XLM-RoBERTa)
* 🧠 Explainable AI (SHAP-based token importance)
* 📊 Confidence scoring
* 📄 Handles long text using chunking
* 🌍 Web interface using Gradio

---

## 🧪 Example

Input:

```
You are stupid
```

Output:

```
Prediction: BULLYING  
Confidence: 0.98  
```
